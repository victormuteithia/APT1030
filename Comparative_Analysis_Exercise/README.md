# Comparative Analysis Exercise: Student Registration Status Checker

## Purpose
This project demonstrates how to implement a simple student registration status checker in four different programming languages: C, Java, JavaScript, and Python. The goal is to compare approaches, syntax, and user interaction in each language while solving the same problem.

## How to Run Each Version

### C (Student_Reg_Status_Checker.c)
1. Open a terminal in this directory.
2. Compile the code:
   ```bash
   gcc Student_Reg_Status_Checker.c -o Student_Reg_Checker
   ```
3. Run the program:
   ```bash
   ./Student_Reg_Checker
   ```

### Java (StudentRegStatusChecker.java)
1. Open a terminal in this directory.
2. Compile the code:
   ```bash
   javac StudentRegStatusChecker.java
   ```
3. Run the program:
   ```bash
   java StudentRegStatusChecker
   ```

### JavaScript (studentRegChecker.js)
1. Make sure you have Node.js installed.
2. Open a terminal in this directory.
3. Run the program:
   ```bash
   node studentRegChecker.js
   ```

### Python (student_reg_checker.py)
1. Make sure you have Python 3 installed.
2. Open a terminal in this directory.
3. Run the program:
   ```bash
   python3 student_reg_checker.py
   ```

## What the Project Demonstrates
- **User Input Handling:** How each language collects and processes user input.
- **Conditional Logic:** How to implement decision-making (if/else) in different languages.
- **Output Formatting:** How to display results and summaries to the user.
- **Cross-language Comparison:** Highlights similarities and differences in syntax, structure, and user experience.

This exercise is ideal for students learning programming fundamentals and for anyone interested in comparing basic program structure across popular languages.
